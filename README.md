#  The Mind Reader: A Python CLI Guessing Game

A logic-driven Command Line Interface (CLI) game that uses **Binary Search** to "read your mind." You pick a number within a range, and the computer finds it in 7 tries or less.

Built as part of my 1st-year journey at **WeThinkCode_** to explore algorithmic efficiency and defensive programming.

##  Features
- **Algorithmic Efficiency:** Uses Binary Search logic to narrow down the correct number.
- **Robust Input Handling:** Custom validation modules to prevent crashes from invalid user inputs.
- **Interactive UX:** Built-in delays and personality-driven terminal prompts for an engaging experience.
- **Environment Aware:** Optimized for Linux/WSL development environments.

##  Tech Stack
- **Language:** Python 3.10+
- **Environment:** Linux / WSL / Windows Terminal
- **Version Control:** Git / GitLab
